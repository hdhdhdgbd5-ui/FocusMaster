# 🎯 FocusMaster: AdSense & Production Setup

To start earning real money from ads, follow these 3 steps:

### 1. Update AdSense ID
Open `index.html` and find all instances of `ca-pub-REPLACE-WITH-YOUR-ID`. 
Replace them with your real Google AdSense Publisher ID.

### 2. Set up Ads.txt
Create a file named `ads.txt` in your root domain and add your publisher info provided by Google.

### 3. Verify on Google Play
To publish this as a "Legendary Android App":
1. Open Android Studio.
2. Use a "WebView" template.
3. Point the URL to your GitHub Pages link.
4. Add your AdSense for Mobile ID.
